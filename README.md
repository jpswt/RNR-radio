# RNR - Rock N Radio

Rock N Radio is a web-based React project created to listen radio broadcasts from around the country.  A user can select various genres and stations to listen to the streaming content. 

## RNR Landing Page

This is the home page for the RNR app.  It allows access for the user to select a radio station of their choice. Every genre has numerous stations that a user can easily access through the pagination of each selected genre.

## RNR Rock Genre Example

This is the Rock Genre home page selection.  It shows all of the stations that are categorized in the "Rock" music category.  Once a user makes a radio selection, the station icon appears in the player and the stream begins to play.
